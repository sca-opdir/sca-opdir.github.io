Créer F6
Modifier F7
Afficher modifications Ctrl+F11
Historique d'un champ Ctrl+F10
Interrompre F12
Activer SIT Ctrl+Maj+F1
Interrompre SIT Ctrl+Maj+F2
Retour (écran précédent) : F3
Atteindre la zone de commande : Ctrl + /
Chercher (dans l'écran d'accueil) : Ctrl + F

Ctrl + Y : mode sélection pour sélectionner et copier du texte en bloc


Parcelle
- Autre parcelle : Maj+F5


Exploitation
- Autre exploitation : Maj+F5

en mode modification : 
- enregistrer : Ctrl+S

dans un champ : 
F1 : aide/informations technique sur le champ
F2 : équivalent au clic (pour les champs clicables)
F4 : aide à la saisie (liste de recherche)

Ctrl + (Touche +) : Ouvre une nouvelle session (une nouvelle fenêtre SAP) s
